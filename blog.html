Important FAQ

What is the Document Object Model(DOM)?

The document object model is a programming interface for html/web documents.It represents the structure of a webpage as a object. We can manipulate Object element dynamically with the help of javaScript.

What is event delegation in the context of the DOM and why is it useful?

Event delegation is technique that we can add a single event to a parent element instead of individual child elements. This is useful because it allows us to manage events more efficiently, particularly in dynamic content where chilren element are to be added or removed.

How do you manipulate an element's attribute and styles using the DOM?

To manipulate an element's attributes we can use element.setAttribute(attribute, value) for set new attribute, element.getAttribute(attribute) to remove attribute, element.style.property='value' to directly modify an inline style, element.classList.add(class) for adding new class, element.classList.remove(className) for remove a class, element.classList.toggle(className).

How do you select an element from the DOM?

We can select from the DOM using various methods like document.getElementById(id), document.getElementsByClassName(className), document.getElementByTagName(tagname), document.querySelector(id/class), document.querySelectorAll(selector)
